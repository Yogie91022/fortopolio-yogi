Project

FINAL PROJECT ON BOARD COMPUTER PROTOTYPE DESIGN FOR POCKETQUBE

Developing a prototype On Board Computer (OBC) system for PocketQube class mini satellites using an Atmega328 microcontroller as the control center. This system integrates an MPU6050 sensor for orientation detection, a LoRa RFM95 module for long-distance communication with the Ground Control Station, and a MicroSD card for mission data storage. This prototype is capable of reading sensor data in real time, receiving commands from the ground station, and recording all data efficiently. This project serves as a proof of concept for a reliable, lightweight, and energy-efficient small satellite control system.

RTC-Based Smart Street Lighting System (PJU Otomatis)

This project aims to design and test an automatic street lighting (PJU) system based on a Real Time Clock (RTC) module that can regulate the on and off times of the lights according to a predetermined schedule. This system is designed to improve energy efficiency and reduce reliance on manual operation. Final Results: Test results show that the system can control the on and off times of the lights automatically and consistently according to the programmed schedule. This project forms the basis for implementing an energy-efficient and easily implemented Smart Street Lighting (PJU) system on both small and large scales.

Obstacle Avoidance with Bluetooth Controller

This project develops a smart mobile robot that is capable of automatically avoiding obstacles using ultrasonic sensors, and can be manually controlled via Bluetooth using a smartphone application.
